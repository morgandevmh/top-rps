# Rock Paper Scissors

A console-based Rock Paper Scissors game built as part of The Odin Project's Foundations Course.

## How to Play

1. Open `index.html` in your browser
2. Open the console (F12)
3. Enter your choice when prompted: rock, paper, or scissors
4. Play 5 rounds against the computer
5. Winner is declared based on final score

## Learning Focus

This project practices:
- Functions and parameters
- Conditional logic
- Variables and scope
- Loops
- String manipulation
